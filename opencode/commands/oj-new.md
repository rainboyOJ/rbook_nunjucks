Create a new rbook-style OJ problem directory.

Arguments: `$ARGUMENTS`

Use the self-contained helper:

```bash
.opencode/oj-tools/bin/oj new $ARGUMENTS
```

After running it, summarize:

- the created problem directory;
- which files were created;
- which files already existed and were skipped;
- the next student action.

Do not write a full solution. The student should first read the statement and restate the problem.
