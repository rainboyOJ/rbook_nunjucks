## 区间二分性质

现有分割方法如下: `m=(l+r)/2 -> new_left = [l,m],new_right = [m+1,r]`


:::colorfulbox

证明1:

任意长度为$n(n \geslant 1)$的区间按上面的分法

1. 最会一定会分割成长度为$1$的区间
2. 最多进行$\lceil log_2^{n} \rceil$次分割
3. 不会有超过$4 \times n$个元素

:::


::: colorfulbox

推论


任意长度为$n(n \geslant 1)$的区间按上面的分法

1. 不会发生无限递归的情况.
2. 分割的同级别区间两两不相交

:::
