## 旧文要点

二叉搜索树满足：

- 每个节点左子树中的值都小于该节点。
- 每个节点右子树中的值都大于该节点。
- 每棵子树也满足同样性质。

常见操作包括查找、插入、删除、最小值、最大值、前驱和后继。正式教程中已经按新的文章结构重新整理这些内容。
