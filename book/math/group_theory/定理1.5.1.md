这是一个关于群论中**元素阶（Order of an Element）**的核心定理（定理 1.5.1）。它描述了元素的阶在取逆、取幂以及乘积运算下的性质。

以下是该定理四个部分的完整证明：

---

### **前置定义**
元素的阶 $\text{ord } a$ 定义为使得 $a^n = e$ 成立的**最小正整数** $n$。如果这样的 $n$ 不存在，则称 $a$ 为无限阶。

---

### **证明 (1)：逆元的阶等于原元素的阶**
**命题：** 对任意 $a \in G$，有 $\text{ord } a = \text{ord } a^{-1}$。

**证明：**
设 $\text{ord } a = n$，$\text{ord } a^{-1} = k$。
1.  **证明 $k \le n$：**
    因为 $a^n = e$，两边取逆：
    $$(a^n)^{-1} = e^{-1} \implies (a^{-1})^n = e$$
    因为 $k$ 是使得 $(a^{-1})^x = e$ 的最小正整数，而这里 $x=n$ 成立，所以 $k \le n$。

2.  **证明 $n \le k$：**
    因为 $\text{ord } a^{-1} = k$，即 $(a^{-1})^k = e$，整理得 $a^{-k} = e$。
    两边同时取逆：
    $$(a^{-k})^{-1} = e^{-1} \implies a^k = e$$
    因为 $n$ 是使得 $a^x = e$ 的最小正整数，而这里 $x=k$ 成立，所以 $n \le k$。

3.  **结论：**
    综上，$n = k$。即 $\text{ord } a = \text{ord } a^{-1}$。
    *(注：若 $a$ 为无限阶，则 $a^{-1}$ 必为无限阶，否则会导致 $a$ 也是有限阶，矛盾。)*

---

### **证明 (2)：若 $a^m = e$，则阶整除 $m$**
**命题：** 设 $\text{ord } a = n$，如果有 $m \in \mathbb{Z}$ 使 $a^m = e$，则 $n \mid m$。

**证明：**
利用**带余除法（Division Algorithm）**。
将 $m$ 除以 $n$，得到商 $q$ 和余数 $r$：
$$m = qn + r, \quad 0 \le r < n$$
代入 $a^m$：
$$a^m = a^{qn + r} = (a^n)^q \cdot a^r$$
已知 $a^m = e$ 且 $a^n = e$，所以：
$$e = e^q \cdot a^r \implies a^r = e$$
**关键点：** $n$ 是使得 $a^x = e$ 的**最小正整数**。
而这里的余数 $r$ 满足 $0 \le r < n$。
* 如果 $r > 0$，则 $r$ 是比 $n$ 更小的正整数且满足 $a^r=e$，这与 $n$ 的最小性矛盾。
* 因此，必须有 $r = 0$。

这意味着 $m = qn$，即 $n$ 整除 $m$ ($n \mid m$)。

---

### **证明 (3)：幂的阶的公式**
**命题：** 设 $\text{ord } a = n$，则 $\text{ord } a^m = \frac{n}{\gcd(n, m)}$。

**证明：**
设 $d = \gcd(n, m)$。我们可以把 $n$ 和 $m$ 写成：
$$n = d n', \quad m = d m'$$
其中 $\gcd(n', m') = 1$。
我们要证明的目标是：$\text{ord } a^m = n'$（因为 $n' = n/d$）。

令 $k = \text{ord } a^m$。我们需要证明 $k = n'$。

1.  **证明 $k \mid n'$：**
    计算 $(a^m)^{n'}$：
    $$(a^m)^{n'} = a^{m n'} = a^{(d m') n'} = a^{m' (d n')} = a^{m' n} = (a^n)^{m'} = e^{m'} = e$$
    根据刚才证明的性质 (2)，既然 $(a^m)^{n'} = e$，那么 $a^m$ 的阶 $k$ 必须整除指数 $n'$。
    即 **$k \mid n'$**。

2.  **证明 $n' \mid k$：**
    因为 $k$ 是 $a^m$ 的阶，所以 $(a^m)^k = e$，即 $a^{mk} = e$。
    根据性质 (2)，$a$ 的阶 $n$ 必须整除 $mk$：
    $$n \mid mk$$
    代入 $n=dn', m=dm'$：
    $$dn' \mid dm'k \implies n' \mid m'k$$
    因为 $\gcd(n', m') = 1$，根据欧几里得引理，必有 **$n' \mid k$**。

**结论：**
既然 $k \mid n'$ 且 $n' \mid k$，且它们都是正整数，所以 $k = n'$。
即 $\text{ord } a^m = \frac{n}{\gcd(n, m)}$。

---

### **证明 (4)：交换元素积的阶**
**命题：** 设 $\text{ord } a = n, \text{ord } b = m$。如果 $ab = ba$ 且 $\gcd(n, m) = 1$，则 $\text{ord}(ab) = mn$。

**证明：**
令 $k = \text{ord}(ab)$。我们需要证明 $k = mn$。

1.  **证明 $k \mid mn$：**
    因为 $a$ 和 $b$ 可交换 ($ab=ba$)，所以 $(ab)^x = a^x b^x$。
    考察 $(ab)^{mn}$：
    $$(ab)^{mn} = a^{mn} b^{mn} = (a^n)^m (b^m)^n = e^m e^n = e \cdot e = e$$
    根据性质 (2)，$k \mid mn$。

2.  **证明 $mn \mid k$：**
    因为 $k$ 是阶，所以 $(ab)^k = e \implies a^k b^k = e \implies a^k = b^{-k}$。
    
    * **证明 $n \mid k$：**
        两边同时取 $m$ 次幂：
        $$(a^k)^m = (b^{-k})^m = (b^m)^{-k} = e^{-k} = e$$
        所以 $a^{km} = e$。根据性质 (2)，$n \mid km$。
        因为 $\gcd(n, m) = 1$，所以 **$n \mid k$**。

    * **证明 $m \mid k$：**
        回到 $a^k = b^{-k}$，两边同时取 $n$ 次幂：
        $$(b^{-k})^n = (a^k)^n = (a^n)^k = e^k = e \implies b^{-kn} = e \implies b^{kn} = e$$
        根据性质 (2)，$m \mid kn$。
        因为 $\gcd(m, n) = 1$，所以 **$m \mid k$**。

    * **合并：**
        因为 $n \mid k$ 且 $m \mid k$，且 $\gcd(n, m) = 1$，所以它们的乘积也整除 $k$：
        **$mn \mid k$**。

**结论：**
由 $k \mid mn$ 和 $mn \mid k$ 可知，$k = mn$。
