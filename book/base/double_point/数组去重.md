## 方法二: 双指针


1. 数组排序,相同的元素会在一起
2. stl的实现`std::unique`函数,可以将相同的元素移动到数组的最后面,返回值是去重后的数组的最后一个元素的位置

两个数组实现
核心思路: 如果当前元素和前一个元素不相同,就表明这个元素是第一不同的元素,那么就将当前元素放入b数组中

```cpp
int a[maxn],b[maxn]
sort(a+1,a+maxn+1);
int idx = 0;
b[++idx] = a[1]; // 将第一个元素放入b数组中
for(int i = 2 ;i< maxn;i++ ){
    // 如果当前元素和前一个元素不相同,那么就将当前元素放入b数组中
    if(a[i] != a[i-1]){
        b[++idx] = a[i];
    }
}
```

使用单个数组

- 指针j: 指向不重复元素的最后一个位置
- 指针i: 用于扫描数组,

核心思想: 当前元素和不重复元素的最后一个元素不相同,那么就将当前元素放入不重复元素的最后一个位置

```cpp
int a[maxn];
sort(a+1,a+maxn+1);
int j = 1;
int idx = 0;
for(int j = 1;i<maxn;i++){
    if(a[j] != a[i]){
        a[++j] = a[i];
    }
}
```
