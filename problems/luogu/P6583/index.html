1 题目简述

题目：给定正整数 nnn (1≤n≤10121 \le n \le 10^{12}1≤n≤1012)，求出有序整数对 (x,y)(x,y)(x,y) 的个数，满足 1≤x,y≤n1\le x,y\le n1≤x,y≤n 且 xy\frac{x}{y}yx​ 可以表示为十进制有限小数。

核心难点：数据规模高达 101210^{12}1012，普通的 O(n)O(n)O(n) 暴力枚举显然会超时，我们需要推导出一个 O(n)O(\sqrt{n})O(n​) 或更优的数论解法。

//是否 y只有因子2 和 5
bool is_only_25(int y) {
    while( y % 2 == 0) y /=2;
    while( y % 5 == 0) y /=5;
    return y == 1;
}

int main () {
    int n;
    std::cin >> n;
    int ans = 0;
    for(int x = 1;x <= n ;++x ) // x: 1->n
    {
        for(int y = 1;y <= n ;++y ) // y: 1->n

            int d = std::gcd(x,y);

                printf("%d / %d \n",x,y);
                ans++;
            }
        }
    }
    std::cout << ans << "\n";
    
    return 0;
}

对于题目中的分数 xy\frac{x}{y}yx​（可能未化简），我们可以将分母 yyy 分解为两部分：

y=k×zy = k \times zy=k×z
其中：

  • zzz 是 yyy 中只包含因子 2 和 5 的部分（即 z=2a×5bz = 2^a \times 5^bz=2a×5b）。
  • kkk 是 yyy 去除 2 和 5 因子后剩下的部分，此时必然满足 gcd⁡(k,10)=1\gcd(k, 10) = 1gcd(k,10)=1。

消除因子 k

为了让 xy=xk×z\frac{x}{y} = \frac{x}{k \times z}yx​=k×zx​ 成为有限小数，分子 xxx 必须能够约掉分母中的 kkk。这意味着 xxx 必须是 kkk 的倍数。 即 x=c×kx = c \times kx=c×k。

计数转化

一个y对应一个k,但是k可能对应多个y. 既然枚举y不行,那我们转而去枚举kkk

我们可以改变枚举对象：不枚举 yyy，而是枚举 kkk。 对于一个固定的 kkk（满足 gcd⁡(k,10)=1\gcd(k, 10) = 1gcd(k,10)=1）：

  1. xxx 的限制：因为 1≤x≤n1 \le x \le n1≤x≤n 且 k∣xk|xk∣x，所以合法的 xxx 有 ⌊nk⌋\lfloor \frac{n}{k} \rfloor⌊kn​⌋ 个。
  2. yyy 的限制：因为 y=k×z≤ny = k \times z \le ny=k×z≤n，所以 z≤⌊nk⌋z \le \lfloor \frac{n}{k} \rfloorz≤⌊kn​⌋。且 zzz 必须形如 2a×5b2^a \times 5^b2a×5b。
    • 设函数 f(m)f(m)f(m) 表示 ≤m\le m≤m 的数中形如 2a×5b2^a \times 5^b2a×5b 的数的个数。
    • 那么合法的 yyy 就对应着 f(⌊nk⌋)f(\lfloor \frac{n}{k} \rfloor)f(⌊kn​⌋) 个合法的 zzz。

综合上述两点，总答案为：

Ans=∑k=1,gcd⁡(k,10)=1n(⌊nk⌋×f(⌊nk⌋))\text{Ans} = \sum_{k=1, \gcd(k,10)=1}^{n} \left( \lfloor \frac{n}{k} \rfloor \times f\left(\lfloor \frac{n}{k} \rfloor\right) \right)Ans=k=1,gcd(k,10)=1∑n​(⌊kn​⌋×f(⌊kn​⌋))

ll f(ll v) {
    return std::upper_bound(a25.begin(), a25.end(),v) - a25.begin();
}

    return n - n/2 - n/5 + n/10;
}

    std::vector<ll> a5;

    for(ll a = 0;a <= 39 ;++a ) // a: 1->39
    {
        for( auto b : a5)
        {
            ll t= (1ull << a) *  b;
            if( t >up) break;
            a25.push_back(t);
        }
    }
    std::sort(a25.begin(),a25.end());

    // for( auto a : a25) std::cout << a << " ";
    // std::cout   << "\n";
}

优化 1：整除分块

观察发现，⌊nk⌋\lfloor \frac{n}{k} \rfloor⌊kn​⌋ 的值呈阶梯状下降。我们可以把 ⌊nk⌋\lfloor \frac{n}{k} \rfloor⌊kn​⌋ 值相同的 kkk 归为一个区间 [l,r][l, r][l,r] 统一计算。

  • 区间右端点计算：r=n/(n/l)r = n / (n / l)r=n/(n/l)。
  • 区间贡献：
    (区间内与10互质的数)×(⌊nl⌋×f(⌊nl⌋)) (\text{区间内与10互质的数}) \times (\lfloor \frac{n}{l} \rfloor \times f(\lfloor \frac{n}{l} \rfloor)) (区间内与10互质的数)×(⌊ln​⌋×f(⌊ln​⌋))

优化 2：快速计算互质个数

如何快速求区间 [l,r][l, r][l,r] 内与 10 互质的数的个数？ 利用前缀和思想：count(r) - count(l-1)。 由于与 10 互质的分布具有周期性（周期为 10，互质数为 1, 3, 7, 9），我们可以 O(1)O(1)O(1) 算出：

count(x)=(x/10)×4+check(x%10)\text{count}(x) = (x / 10) \times 4 + \text{check}(x \% 10)count(x)=(x/10)×4+check(x%10)

        #include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

typedef long long ll;

ll n;
vector<ll> nums; // 存储所有 2^a * 5^b

// 预处理所有 <= 10^12 的 2^a * 5^b
void init() {
    // 2^40 > 10^12, 5^18 > 10^12，范围很小
    for (ll i = 1; i <= n; i *= 2) {
        for (ll j = i; j <= n; j *= 5) {
            nums.push_back(j);
        }
    }
    sort(nums.begin(), nums.end());
}

// 计算 1 到 x 之间与 10 互质的数的个数
ll count_coprime(ll x) {
    ll cycle = x / 10;
    ll rem = x % 10;
    // 周期内互质个数为 4 (1, 3, 7, 9)
    ll ans = cycle * 4;
    
    // 处理余数部分
    // a 数组对应 0-9 范围内与 10 互质数的累积个数
    static int a[10] = {0, 1, 1, 2, 2, 2, 2, 3, 3, 4};
    ans += a[rem];
    
    return ans;
}

// 计算 <= val 的数中，只包含因子 2 和 5 的数的个数
ll get_f(ll val) {
    // 使用 upper_bound 找到第一个 > val 的位置
    // 减去 begin() 得到的就是 <= val 的元素个数
    return upper_bound(nums.begin(), nums.end(), val) - nums.begin();
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);

    if (cin >> n) {
        // 特判 n 特别小的情况，防止 init 出错（虽非必要但严谨）
        if (n == 0) { cout << 0 << endl; return 0; }

        init(); // 预处理

        ll ans = 0;
        // 整除分块
        for (ll l = 1, r; l <= n; l = r + 1) {
            ll v = n / l;
            
            if (v == 0) r = n; // 防止除0错误
            else r = n / v;
            
            // 区间 [l, r] 内 k 的贡献
            ll k_count = count_coprime(r) - count_coprime(l - 1);
            
            // 累加：k的数量 * (v * f(v))
            ans += k_count * (v * get_f(v));
        }

        cout << ans << endl;
    }

    return 0;
}
