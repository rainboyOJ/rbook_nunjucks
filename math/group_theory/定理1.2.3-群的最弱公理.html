这张图片展示了群论中一个非常著名的定理：群定义的“最弱公理”形式。

通常我们定义群时，要求有双边单位元（ea=ae=aea=ae=aea=ae=a）和双边逆元（a′a=aa′=ea'a=aa'=ea′a=aa′=e）。 但这个定理告诉我们：只要有“左单位元”和“左逆元”，再加上结合律，就足以保证它是一个群（即右边的性质会自动成立）。

这就像我们在写代码时做“极简实现”——只要实现了左边的逻辑，右边的逻辑在数学上自动就有了。

下面是这个定理的完整证明。

证明思路

我们需要证明两个方向（充分必要条件）：

  1. 必要性 (⇒\Rightarrow⇒)：如果 GGG 是群，那么 (1)(2)(3) 成立。

    • 这很简单，因为群的定义本身就包含了双边单位元和双边逆元，自然也就满足左边的。

  2. 充分性 (⇐\Leftarrow⇐)：如果 (1)(2)(3) 成立，那么 GGG 是群。

    • 这是证明的核心。我们需要利用已知的“左性质”，推导出“右逆元”和“右单位元”。

充分性证明步骤

假设条件 (1) 结合律，(2) 左单位元 eee，(3) 左逆元 a′a'a′ 都成立。

第一步：证明“左逆元”也是“右逆元”

目标： 证明 aa′=eaa' = eaa′=e。

  1. 根据条件 (3)，对于 a∈Ga \in Ga∈G，存在左逆元 a′a'a′，使得 a′a=ea'a = ea′a=e。
  2. 同样的，对于这个 a′a'a′，它也是 GGG 中的元素，所以它也有它自己的左逆元。我们把 a′a'a′ 的左逆元记作 a′′a''a′′。 也就是：a′′a′=ea''a' = ea′′a′=e。
  3. 现在我们利用结合律来考察 a′(aa′)a'(aa')a′(aa′)：
    a′(aa′)=(a′a)a′=ea′=a′a'(aa') = (a'a)a' = ea' = a'a′(aa′)=(a′a)a′=ea′=a′
    （这里利用了 a′a=ea'a=ea′a=e 和左单位元性质 ea′=a′ea'=a'ea′=a′）
  4. 现在我们有了等式：a′(aa′)=a′a'(aa') = a'a′(aa′)=a′。 我们在等式两边的最左侧同时乘以 a′′a''a′′：
    a′′(a′(aa′))=a′′a′a''(a'(aa')) = a''a'a′′(a′(aa′))=a′′a′
  5. 利用结合律：
    (a′′a′)(aa′)=a′′a′(a''a')(aa') = a''a'(a′′a′)(aa′)=a′′a′
  6. 因为 a′′a′=ea''a' = ea′′a′=e，代入得：
    e(aa′)=ee(aa') = ee(aa′)=e
  7. 因为 eee 是左单位元 (ex=xex=xex=x)，所以：
    aa′=eaa' = eaa′=e

结论1： a′a'a′ 不仅是 aaa 的左逆元，也是 右逆元。

第二步：证明“左单位元”也是“右单位元”

目标： 证明 ae=aae = aae=a。

  1. 我们计算 aeaeae：
    ae=a(a′a)ae = a(a'a)ae=a(a′a)
    （因为刚才第一步已经证明了 a′a=ea'a=ea′a=e，这是已知条件，其实第一步证明的是 aa′=eaa'=eaa′=e，这里直接用定义 e=a′ae=a'ae=a′a 即可） 等一下，这里要小心，我们要利用刚刚证出来的右逆元结论。
  2. 更严谨的推导：
    ae=a(a′a)ae = a(a'a)ae=a(a′a)
    这里 eee 替换成 a′aa'aa′a 是题目给的条件 (3)。
  3. 利用结合律：
    ae=(aa′)aae = (aa')aae=(aa′)a
  4. 利用第一步刚证出来的结论 aa′=eaa' = eaa′=e：
    ae=eaae = eaae=ea
  5. 利用题目给的条件 (2) 左单位元 ea=aea = aea=a：
    ae=aae = aae=a

结论2： eee 不仅是左单位元，也是 右单位元。

总结

既然我们证明了：

  1. 运算满足结合律（已知）。
  2. eee 既是左单位元，又是右单位元（即 ea=ae=aea = ae = aea=ae=a）。
  3. a′a'a′ 既是左逆元，又是右逆元（即 a′a=aa′=ea'a = aa' = ea′a=aa′=e）。

所以，集合 GGG 满足群的所有标准定义。 证毕。

给你的思考题（这也是算法里常见的思维）：

如果把题目里的“左”全部换成“右”（右单位元 + 右逆元），结论成立吗？ （答案是肯定的，证明过程几乎是“镜像”对称的。）

但是，如果是“左单位元”加上“右逆元”，能构成群吗？（答案是不能，这是一个非常经典的陷阱！）
