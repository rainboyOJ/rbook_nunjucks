这是一个关于群论中**元素阶（Order of an Element）**的核心定理（定理 1.5.1）。它描述了元素的阶在取逆、取幂以及乘积运算下的性质。

以下是该定理四个部分的完整证明：

前置定义

元素的阶 ord a\text{ord } aord a 定义为使得 an=ea^n = ean=e 成立的最小正整数 nnn。如果这样的 nnn 不存在，则称 aaa 为无限阶。

证明 (1)：逆元的阶等于原元素的阶

命题： 对任意 a∈Ga \in Ga∈G，有 ord a=ord a−1\text{ord } a = \text{ord } a^{-1}ord a=ord a−1。

证明： 设 ord a=n\text{ord } a = nord a=n，ord a−1=k\text{ord } a^{-1} = kord a−1=k。

  1. 证明 k≤nk \le nk≤n： 因为 an=ea^n = ean=e，两边取逆：

    (an)−1=e−1  ⟹  (a−1)n=e(a^n)^{-1} = e^{-1} \implies (a^{-1})^n = e(an)−1=e−1⟹(a−1)n=e
    因为 kkk 是使得 (a−1)x=e(a^{-1})^x = e(a−1)x=e 的最小正整数，而这里 x=nx=nx=n 成立，所以 k≤nk \le nk≤n。

  2. 证明 n≤kn \le kn≤k： 因为 ord a−1=k\text{ord } a^{-1} = kord a−1=k，即 (a−1)k=e(a^{-1})^k = e(a−1)k=e，整理得 a−k=ea^{-k} = ea−k=e。 两边同时取逆：

    (a−k)−1=e−1  ⟹  ak=e(a^{-k})^{-1} = e^{-1} \implies a^k = e(a−k)−1=e−1⟹ak=e
    因为 nnn 是使得 ax=ea^x = eax=e 的最小正整数，而这里 x=kx=kx=k 成立，所以 n≤kn \le kn≤k。

  3. 结论： 综上，n=kn = kn=k。即 ord a=ord a−1\text{ord } a = \text{ord } a^{-1}ord a=ord a−1。 (注：若 aaa 为无限阶，则 a−1a^{-1}a−1 必为无限阶，否则会导致 aaa 也是有限阶，矛盾。)

证明 (2)：若 am=ea^m = eam=e，则阶整除 mmm

命题： 设 ord a=n\text{ord } a = nord a=n，如果有 m∈Zm \in \mathbb{Z}m∈Z 使 am=ea^m = eam=e，则 n∣mn \mid mn∣m。

证明： 利用带余除法（Division Algorithm）。 将 mmm 除以 nnn，得到商 qqq 和余数 rrr：

m=qn+r,0≤r<nm = qn + r, \quad 0 \le r < nm=qn+r,0≤r<n
代入 ama^mam：
am=aqn+r=(an)q⋅ara^m = a^{qn + r} = (a^n)^q \cdot a^ram=aqn+r=(an)q⋅ar
已知 am=ea^m = eam=e 且 an=ea^n = ean=e，所以：
e=eq⋅ar  ⟹  ar=ee = e^q \cdot a^r \implies a^r = ee=eq⋅ar⟹ar=e
关键点： nnn 是使得 ax=ea^x = eax=e 的最小正整数。 而这里的余数 rrr 满足 0≤r<n0 \le r < n0≤r<n。

  • 如果 r>0r > 0r>0，则 rrr 是比 nnn 更小的正整数且满足 ar=ea^r=ear=e，这与 nnn 的最小性矛盾。
  • 因此，必须有 r=0r = 0r=0。

这意味着 m=qnm = qnm=qn，即 nnn 整除 mmm (n∣mn \mid mn∣m)。

证明 (3)：幂的阶的公式

命题： 设 ord a=n\text{ord } a = nord a=n，则 ord am=ngcd⁡(n,m)\text{ord } a^m = \frac{n}{\gcd(n, m)}ord am=gcd(n,m)n​。

证明： 设 d=gcd⁡(n,m)d = \gcd(n, m)d=gcd(n,m)。我们可以把 nnn 和 mmm 写成：

n=dn′,m=dm′n = d n', \quad m = d m'n=dn′,m=dm′
其中 gcd⁡(n′,m′)=1\gcd(n', m') = 1gcd(n′,m′)=1。 我们要证明的目标是：ord am=n′\text{ord } a^m = n'ord am=n′（因为 n′=n/dn' = n/dn′=n/d）。

令 k=ord amk = \text{ord } a^mk=ord am。我们需要证明 k=n′k = n'k=n′。

  1. 证明 k∣n′k \mid n'k∣n′： 计算 (am)n′(a^m)^{n'}(am)n′：

    (am)n′=amn′=a(dm′)n′=am′(dn′)=am′n=(an)m′=em′=e(a^m)^{n'} = a^{m n'} = a^{(d m') n'} = a^{m' (d n')} = a^{m' n} = (a^n)^{m'} = e^{m'} = e(am)n′=amn′=a(dm′)n′=am′(dn′)=am′n=(an)m′=em′=e
    根据刚才证明的性质 (2)，既然 (am)n′=e(a^m)^{n'} = e(am)n′=e，那么 ama^mam 的阶 kkk 必须整除指数 n′n'n′。 即 k∣n′k \mid n'k∣n′。

  2. 证明 n′∣kn' \mid kn′∣k： 因为 kkk 是 ama^mam 的阶，所以 (am)k=e(a^m)^k = e(am)k=e，即 amk=ea^{mk} = eamk=e。 根据性质 (2)，aaa 的阶 nnn 必须整除 mkmkmk：

    n∣mkn \mid mkn∣mk
    代入 n=dn′,m=dm′n=dn', m=dm'n=dn′,m=dm′：
    dn′∣dm′k  ⟹  n′∣m′kdn' \mid dm'k \implies n' \mid m'kdn′∣dm′k⟹n′∣m′k
    因为 gcd⁡(n′,m′)=1\gcd(n', m') = 1gcd(n′,m′)=1，根据欧几里得引理，必有 n′∣kn' \mid kn′∣k。

结论： 既然 k∣n′k \mid n'k∣n′ 且 n′∣kn' \mid kn′∣k，且它们都是正整数，所以 k=n′k = n'k=n′。 即 ord am=ngcd⁡(n,m)\text{ord } a^m = \frac{n}{\gcd(n, m)}ord am=gcd(n,m)n​。

证明 (4)：交换元素积的阶

命题： 设 ord a=n,ord b=m\text{ord } a = n, \text{ord } b = mord a=n,ord b=m。如果 ab=baab = baab=ba 且 gcd⁡(n,m)=1\gcd(n, m) = 1gcd(n,m)=1，则 ord(ab)=mn\text{ord}(ab) = mnord(ab)=mn。

证明： 令 k=ord(ab)k = \text{ord}(ab)k=ord(ab)。我们需要证明 k=mnk = mnk=mn。

  1. 证明 k∣mnk \mid mnk∣mn： 因为 aaa 和 bbb 可交换 (ab=baab=baab=ba)，所以 (ab)x=axbx(ab)^x = a^x b^x(ab)x=axbx。 考察 (ab)mn(ab)^{mn}(ab)mn：

    (ab)mn=amnbmn=(an)m(bm)n=emen=e⋅e=e(ab)^{mn} = a^{mn} b^{mn} = (a^n)^m (b^m)^n = e^m e^n = e \cdot e = e(ab)mn=amnbmn=(an)m(bm)n=emen=e⋅e=e
    根据性质 (2)，k∣mnk \mid mnk∣mn。

  2. 证明 mn∣kmn \mid kmn∣k： 因为 kkk 是阶，所以 (ab)k=e  ⟹  akbk=e  ⟹  ak=b−k(ab)^k = e \implies a^k b^k = e \implies a^k = b^{-k}(ab)k=e⟹akbk=e⟹ak=b−k。

    • 证明 n∣kn \mid kn∣k： 两边同时取 mmm 次幂：

      (ak)m=(b−k)m=(bm)−k=e−k=e(a^k)^m = (b^{-k})^m = (b^m)^{-k} = e^{-k} = e(ak)m=(b−k)m=(bm)−k=e−k=e
      所以 akm=ea^{km} = eakm=e。根据性质 (2)，n∣kmn \mid kmn∣km。 因为 gcd⁡(n,m)=1\gcd(n, m) = 1gcd(n,m)=1，所以 n∣kn \mid kn∣k。

    • 证明 m∣km \mid km∣k： 回到 ak=b−ka^k = b^{-k}ak=b−k，两边同时取 nnn 次幂：

      (b−k)n=(ak)n=(an)k=ek=e  ⟹  b−kn=e  ⟹  bkn=e(b^{-k})^n = (a^k)^n = (a^n)^k = e^k = e \implies b^{-kn} = e \implies b^{kn} = e(b−k)n=(ak)n=(an)k=ek=e⟹b−kn=e⟹bkn=e
      根据性质 (2)，m∣knm \mid knm∣kn。 因为 gcd⁡(m,n)=1\gcd(m, n) = 1gcd(m,n)=1，所以 m∣km \mid km∣k。

    • 合并： 因为 n∣kn \mid kn∣k 且 m∣km \mid km∣k，且 gcd⁡(n,m)=1\gcd(n, m) = 1gcd(n,m)=1，所以它们的乘积也整除 kkk： mn∣kmn \mid kmn∣k。

结论： 由 k∣mnk \mid mnk∣mn 和 mn∣kmn \mid kmn∣k 可知，k=mnk = mnk=mn。
