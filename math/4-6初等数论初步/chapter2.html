同余定义

同余定义

一般地，设nnn为正整数，aaa 和 bbb 为整数。

  • 如果 aaa 和 bbb 被 nnn 除后余数相同，那么称 aaa 和bbb 模 nnn 同余，记作 a≡b(modn)a\equiv b \pmod na≡b(modn).
  • 若 aaa 和 bbb 被 nnn 除后余数不同，则称aaa 和bbb 模nnn 不同余，记作: a≢b(modn)a \not \equiv b \pmod na≡b(modn)

这是为您重写的证明，采用了极简的算式推导流。

这个定理非常实用，它打通了“余数世界”和“整除世界”的桥梁。我根据其核心性质，将其命名为**“同余差分律”**。

同余差分律 (同余定义的等价形式)

核心直觉：如果两个数 aaa 和 bbb 在圆环跑道上停在同一个位置（同余），那么它们跑过的路程之差，一定是跑道长度 nnn 的整数倍。

a≡b(modn)  ⟺  n∣(a−b)a \equiv b \pmod n \iff n \mid (a - b)a≡b(modn)⟺n∣(a−b)

  1. 定义（带余除法）： 设 a,ba, ba,b 被 nnn 除，商为 q,q′q, q'q,q′，余数为 r,r′r, r'r,r′。

    a=nq+ra = nq + ra=nq+r
    b=nq′+r′b = nq' + r'b=nq′+r′

  2. 做差（构建联系）： 两式相减：

    a−b=n(q−q′)+(r−r′)a - b = n(q - q') + (r - r')a−b=n(q−q′)+(r−r′)

  3. 正向推导 (⇒\Rightarrow⇒)：

    • 若 a≡b(modn)a \equiv b \pmod na≡b(modn)（即余数相同 r=r′r = r'r=r′）
    • ⇒r−r′=0\Rightarrow r - r' = 0⇒r−r′=0
    • 代入差分式： ⇒a−b=n(q−q′)\Rightarrow a - b = n(q - q')⇒a−b=n(q−q′)
    • 结论：n∣(a−b)n \mid (a - b)n∣(a−b)

  4. 逆向推导 (⇐\Leftarrow⇐)：

    • 若 n∣(a−b)n \mid (a - b)n∣(a−b)
    • 观察差分式 a−b=n(q−q′)⏟能被n整除+(r−r′)a - b = \underbrace{n(q - q')}_{\text{能被n整除}} + (r - r')a−b=能被n整除n(q−q′)​​+(r−r′)
    • ⇒n\Rightarrow n⇒n 必须整除剩余部分 (r−r′)(r - r')(r−r′)

    关键限制：

    • 余数差的绝对值必然小于除数：−n<r−r′<n-n < r - r' < n−n<r−r′<n
    • 在此范围内，能被 nnn 整除的数只有一个：000。 ∴r−r′=0⇒r=r′\therefore r - r' = 0 \Rightarrow r = r'∴r−r′=0⇒r=r′ 结论：a≡b(modn)a \equiv b \pmod na≡b(modn)

逻辑：两数相减，正好把“多出来的余数”抵消掉，只剩下 nnn 的整数倍。

根据上面的定义,很容易得到同余的性质

同余的性质

  • 反身性：a≡a(modn)a \equiv a \pmod na≡a(modn)
  • 对称性：若 a≡b(modn)a \equiv b \pmod na≡b(modn)，则 b≡a(modn)b \equiv a \pmod nb≡a(modn)
  • 传递性：若 a≡b(modn)a \equiv b \pmod na≡b(modn) 且 b≡c(modn)b \equiv c \pmod nb≡c(modn)，则 a≡c(modn)a \equiv c \pmod na≡c(modn)

传递性的简易证明:

  • 由 a≡b(modn)a \equiv b \pmod na≡b(modn) 和 b≡c(modn)b \equiv c \pmod nb≡c(modn)，可得 n∣(a−b)n \mid (a - b)n∣(a−b) 和 n∣(b−c)n \mid (b - c)n∣(b−c)
  • 由整除的性质，可得 n∣(a−b+b−c)=n∣(a−c)n \mid (a - b + b - c) = n \mid (a - c)n∣(a−b+b−c)=n∣(a−c)
  • 由同余的定义，可得 a≡c(modn)a \equiv c \pmod na≡c(modn)

同余的运算性质

同余的运算性质

若 a≡b(modn)a \equiv b \pmod na≡b(modn)，c≡d(modn)c \equiv d \pmod nc≡d(modn)，则

  • 加法： a+c≡b+d(modn)a + c \equiv b + d \pmod na+c≡b+d(modn)
  • 减法： a−c≡b−d(modn)a - c \equiv b - d \pmod na−c≡b−d(modn)
  • 乘法： a×c≡b×d(modn)a \times c \equiv b \times d \pmod na×c≡b×d(modn)
  • 倍增: ka≡kb(modn)ka \equiv kb \pmod nka≡kb(modn)
  • 幂运算: am≡bm(modn)a^m \equiv b^m \pmod nam≡bm(modn)

同余消去率

若 ab≡ac(modn)ab \equiv ac \pmod nab≡ac(modn) 且 (a,n)=1(a, n) = 1(a,n)=1，则 b≡c(modn)b \equiv c \pmod nb≡c(modn)

我们统一使用 同余差分律 作为解题万能钥匙： 定义：A≡B(modn)  ⟺  n∣(A−B)A \equiv B \pmod n \iff n \mid (A - B)A≡B(modn)⟺n∣(A−B)。

设 a−b=nk,c−d=nma - b = nk, \quad c - d = nma−b=nk,c−d=nm。

1. 加减法

目标：(a±c)≡(b±d)(modn)(a \pm c) \equiv (b \pm d) \pmod n(a±c)≡(b±d)(modn)

  1. 做差： (a±c)−(b±d)=(a−b)±(c−d)(a \pm c) - (b \pm d) = (a - b) \pm (c - d)(a±c)−(b±d)=(a−b)±(c−d)
  2. 代入： =nk±nm= nk \pm nm=nk±nm =n(k±m)= n(k \pm m)=n(k±m)
  3. 结论： nnn 整除差值 ⇒\Rightarrow⇒ 成立。

2. 乘法

目标：ac≡bd(modn)ac \equiv bd \pmod nac≡bd(modn)

  1. 做差： ac−bdac - bdac−bd
  2. 搭桥 (关键技巧：加一项减一项 bcbcbc)： =ac−bc+bc−bd= ac - \mathbf{bc} + \mathbf{bc} - bd=ac−bc+bc−bd =c(a−b)+b(c−d)= c(a - b) + b(c - d)=c(a−b)+b(c−d)
  3. 代入： =c(nk)+b(nm)= c(nk) + b(nm)=c(nk)+b(nm) =n(ck+bm)= n(ck + bm)=n(ck+bm)
  4. 结论： nnn 整除差值 ⇒\Rightarrow⇒ 成立。

  2. 提取： =k(a−b)= k(a - b)=k(a−b) =k(nk)= k(nk)=k(nk)
  3. 结论： 结果仍是 nnn 的倍数 ⇒\Rightarrow⇒ 成立。

4. 幂运算

目标：am≡bm(modn)a^m \equiv b^m \pmod nam≡bm(modn)

  1. 做差： am−bma^m - b^mam−bm
  2. 因式分解公式： =(a−b)(am−1+am−2b+⋯+bm−1)= (a - b)(a^{m-1} + a^{m-2}b + \dots + b^{m-1})=(a−b)(am−1+am−2b+⋯+bm−1)
  3. 判定： ∵a≡b⇒n∣(a−b)\because a \equiv b \Rightarrow n \mid (a - b)∵a≡b⇒n∣(a−b) ∴n\therefore n∴n 必整除包含 (a−b)(a - b)(a−b) 因子的整个乘积。
  4. 结论： 成立。

5. 同余消去律

目标：若 ab≡ac(modn)ab \equiv ac \pmod nab≡ac(modn) 且 (a,n)=1(a, n) = 1(a,n)=1，则 b≡cb \equiv cb≡c。

  1. 转化： ab≡ac⇒n∣(ab−ac)ab \equiv ac \Rightarrow n \mid (ab - ac)ab≡ac⇒n∣(ab−ac) ⇒n∣a(b−c)\Rightarrow n \mid a(b - c)⇒n∣a(b−c)
  2. 引用互质消去律： ∵n∣a(b−c)\because n \mid a(b - c)∵n∣a(b−c) 且 (n,a)=1(n, a) = 1(n,a)=1 ∴n\therefore n∴n 无法“依附”于 aaa，必须整除 (b−c)(b - c)(b−c)。
  3. 结论： n∣(b−c)⇒b≡c(modn)n \mid (b - c) \Rightarrow b \equiv c \pmod nn∣(b−c)⇒b≡c(modn)。

核心技巧：

  • 加减法：直接做差。
  • 乘法：搭桥法（加一项减一项）。
  • 倍增：提取公因子。
  • 幂运算：因式分解。
  • 消去律：利用互质性质。

这些性质让同余运算像代数中的等式一样自由操作，是解题的强大工具。

6. 同余消去律 (通用版)

这个定理告诉我们：在同余式两边同时消去一个数时，模数也必须同时除以它们的最大公约数。

同余消去律 (通用版)

ca≡cb(modn)  ⟺  a≡b(modn(c,n)) ca \equiv cb \pmod n \iff a \equiv b \pmod{\frac{n}{(c,n)}} ca≡cb(modn)⟺a≡b(mod(c,n)n​)

证明过程

设 d=(c,n)d = (c, n)d=(c,n) 为 ccc 和 nnn 的最大公约数。

  1. 转化 (同余转整除)：

    • ca≡cb(modn)ca \equiv cb \pmod nca≡cb(modn)   ⟺  n∣c(a−b)\iff n \mid c(a - b)⟺n∣c(a−b)

  2. 约分 (提取公约数 ddd)：

    • 将 nnn 写为 d⋅ndd \cdot \frac{n}{d}d⋅dn​，将 ccc 写为 d⋅cdd \cdot \frac{c}{d}d⋅dc​   ⟺  d⋅nd∣d⋅cd(a−b)\iff d \cdot \frac{n}{d} \mid d \cdot \frac{c}{d} (a - b)⟺d⋅dn​∣d⋅dc​(a−b)

  3. 化简 (两边同除以 ddd)：

    •   ⟺  nd∣cd(a−b)\iff \frac{n}{d} \mid \frac{c}{d} (a - b)⟺dn​∣dc​(a−b)

  4. 引用 (互质消去律)：

    • ∵\because∵ 除去最大公约数后，(nd,cd)=1(\frac{n}{d}, \frac{c}{d}) = 1(dn​,dc​)=1 (互质)
    • 根据互质消去律，nd\frac{n}{d}dn​ 无法整除 cd\frac{c}{d}dc​，必须整除后半部分
    •   ⟺  nd∣(a−b)\iff \frac{n}{d} \mid (a - b)⟺dn​∣(a−b)

  5. 结论 (整除转同余)：   ⟺  a≡b(modnd)\iff a \equiv b \pmod{\frac{n}{d}}⟺a≡b(moddn​) 即 a≡b(modn(c,n))a \equiv b \pmod{\frac{n}{(c,n)}}a≡b(mod(c,n)n​)

证毕

直觉模型: 因为乘数 ccc 里面含有模数 nnn 的“基因”（公因子），所以在消去 ccc 之后，模数 nnn 必须把这部分“基因”也剔除掉，剩下的才是 aaa 和 bbb 真正的约束。

举个栗子 🌰

假设 模数 n=12n = 12n=12（你的目标是凑齐 12 的倍数）。 我们看方程：6a≡6b(mod12)6a \equiv 6b \pmod{12}6a≡6b(mod12)。 这里 c=6c = 6c=6。

  • 目标：6×(a−b)6 \times (a-b)6×(a−b) 必须是 121212 的倍数。
  • 分析 ccc (即 6)：
    • 666 里面已经包含了因子 222 和 333。
    • 121212 里面包含了因子 2,2,32, 2, 32,2,3。
    • 你看，ccc 已经提供了 121212 所需的大部分因子，只缺一个 222 了！
  • 结果：
    • (a−b)(a-b)(a−b) 只需要负责提供剩下的那个 222 就可以了。
    • 所以，aaa 和 bbb 的差距只要是 222 的倍数 就行，不需要是 121212 的倍数。
    • 这就是为什么模数从 121212 变成了 222 (12(6,12)=2\frac{12}{(6,12)} = 2(6,12)12​=2)。

7. 同倍律

a≡b(modm)→ka≡kb(modk)m a \equiv b \pmod m \to ka \equiv kb \pmod km a≡b(modm)→ka≡kb(modk)m

剩余类及其运算

在此之前，我们学习了同余 a≡b(modm)a \equiv b \pmod ma≡b(modm)，这意味着 aaa 和 bbb 除以 mmm 的余数相同。 现在，我们将所有**“同余的数”打包放进同一个袋子里，这个袋子就叫剩余类**。

  • 定义：对于给定的模 mmm，我们可以根据余数是 0,1,2,…,m−10, 1, 2, \dots, m-10,1,2,…,m−1 将所有的整数划分为 mmm 个集合。
  • 例子 (模 3)： 如果我们模 333，整数被分为 3 类：
    • 余 0 的类 (0‾\overline{0}0)：{…,−6,−3,0,3,6,… }\{\dots, -6, -3, 0, 3, 6, \dots\}{…,−6,−3,0,3,6,…}
    • 余 1 的类 (1‾\overline{1}1)：{…,−5,−2,1,4,7,… }\{\dots, -5, -2, 1, 4, 7, \dots\}{…,−5,−2,1,4,7,…}
    • 余 2 的类 (2‾\overline{2}2)：{…,−4,−1,2,5,8,… }\{\dots, -4, -1, 2, 5, 8, \dots\}{…,−4,−1,2,5,8,…}
  • 符号：
    • 模 mmm 的剩余类集合记作 ZmZ_mZm​。
    • 例如 Z3={0‾,1‾,2‾}Z_3 = \{\overline{0}, \overline{1}, \overline{2}\}Z3​={0,1,2}。
    • 其中的每一个元素（如 1‾\overline{1}1）代表的不仅仅是数字 1，而是所有除以 3 余 1 的整数的集合。

有了这些“类”，我们就可以直接对它们进行加法和乘法运算。这不再是普通数字的运算，而是集合与集合之间的运算。

运算规则

  • 加法：a‾+b‾=a+b‾\overline{a} + \overline{b} = \overline{a+b}a+b=a+b​
  • 乘法：a‾×b‾=a×b‾\overline{a} \times \overline{b} = \overline{a \times b}a×b=a×b​

核心思想：你可以从类 a‾\overline{a}a 中随便拿一个代表数，从类 b‾\overline{b}b 中随便拿一个代表数，算完之后看结果属于哪个类。结果与你选哪个代表数无关（这叫运算的“良定义”）。

证明剩余类的运算与代表数的选择无关，可以通过同余的性质来证明。

  • 加法： 设 a≡a′(modm)a \equiv a' \pmod ma≡a′(modm)，b≡b′(modm)b \equiv b' \pmod mb≡b′(modm)，则

    a+b≡a′+b′(modm)a + b \equiv a' + b' \pmod ma+b≡a′+b′(modm)
    因此，a‾+b‾=a+b‾=a′+b′‾=a′‾+b′‾\overline{a} + \overline{b} = \overline{a + b} = \overline{a' + b'} = \overline{a'} + \overline{b'}a+b=a+b​=a′+b′​=a′+b′。

  • 乘法： 类似地，可以证明 a‾×b‾=a′‾×b′‾\overline{a} \times \overline{b} = \overline{a'} \times \overline{b'}a×b=a′×b′。

这里是为你重写的证明，完全采用了你要的“极简算式推导流”风格。

模 n 逆元存在充要条件

核心逻辑：将抽象的“模运算”降维成具体的“整系数方程”，利用 裴蜀定理 在“方程有解”与“互质”之间建立双向通道。

  1. 转化：定义 →\to→ 方程

    • [a][a][a] 有逆元   ⟺  [a][x]=[1]  ⟺  [ax]=[1]\iff [a][x] = [1] \iff [ax] = [1]⟺[a][x]=[1]⟺[ax]=[1]
    •   ⟺  ax≡1(modn)\iff ax \equiv 1 \pmod n⟺ax≡1(modn)
    •   ⟺  n∣1−ax\iff n \mid 1 - ax⟺n∣1−ax
    •   ⟺  ny=1−ax→ny+ax=1\iff ny = 1 - ax \to ny + ax = 1⟺ny=1−ax→ny+ax=1
    •   ⟺  ∃x,y∈Z\iff \exists x, y \in \mathbb{Z}⟺∃x,y∈Z，使得 ax+ny=1ax + ny = 1ax+ny=1

  2. 必要性 (⇒\Rightarrow⇒)：方程 →\to→ 互质

    • 令 d=(a,n)d = (a, n)d=(a,n)
    • ∵d∣a\because d \mid a∵d∣a 且 d∣n⇒d∣(ax+ny)d \mid n \Rightarrow d \mid (ax + ny)d∣n⇒d∣(ax+ny)
    • 代入方程 ax+ny=1⇒d∣1ax + ny = 1 \Rightarrow d \mid 1ax+ny=1⇒d∣1
    • ∴(a,n)=1\therefore (a, n) = 1∴(a,n)=1

  3. 充分性 (⇐\Leftarrow⇐)：互质 →\to→ 方程 →\to→ 逆元

    • 若 (a,n)=1(a, n) = 1(a,n)=1

零因子

图片中展示了模 4 (Z4Z_4Z4​) 和模 5 (Z5Z_5Z5​) 的加法与乘法表，对比这两个表能发现非常重要的性质。

3.1 模 4 的运算 (Z4Z_4Z4​)

  • 集合：{0‾,1‾,2‾,3‾}\{\overline{0}, \overline{1}, \overline{2}, \overline{3}\}{0,1,2,3}
  • 特殊的现象： 在乘法表中，我们发现：
    2‾×2‾=4‾=0‾\overline{2} \times \overline{2} = \overline{4} = \overline{0}2×2=4=0
    这意味着：两个非零的数相乘，结果竟然变成了 0！
    • 这种现象在普通算术中是不存在的，但在模运算（特别是模合数）中很常见。这些非零但乘积为零的数被称为“零因子”。

3.2 模 5 的运算 (Z5Z_5Z5​)

  • 集合：{0‾,1‾,2‾,3‾,4‾}\{\overline{0}, \overline{1}, \overline{2}, \overline{3}, \overline{4}\}{0,1,2,3,4}
  • 性质： 由于 5 是素数，观察其乘法表可以发现：
    • 没有出现“两个非零数相乘等于 0”的情况。
    • 这使得 Z5Z_5Z5​ 的结构比 Z4Z_4Z4​ 更“完美”，性质更接近我们熟悉的实数运算。

总结与升华

  • 抽象化：剩余类把无穷多的整数简化成了有限个“对象” (000 到 m−1m-1m−1)。
  • 运算封闭：在 ZmZ_mZm​ 里怎么算（加或乘），结果永远还在 ZmZ_mZm​ 里，跑不出去。
  • 素数的特殊性：对比模 4 和模 5 的运算表暗示了：模是素数时，运算性质通常比模是合数时更优良（没有零因子）。

费马小定理

这是模运算中关于素数的一个神奇性质。

定理描述

设 ppp 为素数，对于任意整数 aaa，有：

ap≡a(modp)a^p \equiv a \pmod pap≡a(modp)

常用推论（更常用的形式）： 如果 aaa 不是 ppp 的倍数（即 (a,p)=1(a, p) = 1(a,p)=1），两边消去一个 aaa 得到：

ap−1≡1(modp)a^{p-1} \equiv 1 \pmod pap−1≡1(modp)

核心直觉

在模素数 ppp 的世界里，如果你把一个数 aaa 连续乘 p−1p-1p−1 次，它就会“归一”，变回 111。这定义了一个长度为 p−1p-1p−1 的循环周期。

证明

核心逻辑：构造两组数，一组是“原版”，一组是“放大版（乘 aaa）”。利用模运算的“重排”性质建立乘积等式，最后像解方程一样消去阶乘。

  1. 构造：取 111 到 p−1p-1p−1 的整数集 SSS

    • S={1,2,…,p−1}S = \{1, 2, \dots, p-1\}S={1,2,…,p−1}
    • 将 SSS 中各项同乘 aaa 得到 S′S'S′
    • S′={a,2a,…,(p−1)a}S' = \{a, 2a, \dots, (p-1)a\}S′={a,2a,…,(p−1)a}

  2. 性质：同余重排

    • 互不相同：若 i⋅a≡j⋅a(modp)i\cdot a \equiv j\cdot a \pmod pi⋅a≡j⋅a(modp)，根据同余消去律（因为 (a,p)=1(a,p)=1(a,p)=1），必有 i≡ji \equiv ji≡j。
      • 这里用到了p→q  ⟺  ¬q→¬pp \to q \iff \neg q \to \neg pp→q⟺¬q→¬p
    • 非零：因为 ppp 是素数且 aaa 不是 ppp 的倍数，所以 k⋅ak \cdot ak⋅a 模 ppp 不可能为 0。
    • 结论：S′S'S′ 中的元素模 ppp 之后，刚好就是 SSS 中的元素（只是顺序打乱了）。

  3. 做积：两集合元素连乘，积必同余

    • ∏(S′)≡∏(S)(modp)\prod (S') \equiv \prod (S) \pmod p∏(S′)≡∏(S)(modp)
    • ⇒a⋅2a⋯(p−1)a≡1⋅2⋯(p−1)(modp)\Rightarrow a \cdot 2a \cdots (p-1)a \equiv 1 \cdot 2 \cdots (p-1) \pmod p⇒a⋅2a⋯(p−1)a≡1⋅2⋯(p−1)(modp)

  4. 提取：整理算式

    • 左边提取 p−1p-1p−1 个 aaa，并将数字部分合并为阶乘：
    • ap−1⋅(p−1)!≡(p−1)!(modp)a^{p-1} \cdot (p-1)! \equiv (p-1)! \pmod pap−1⋅(p−1)!≡(p−1)!(modp)

  5. 消去：同余消去律

    • ∵p\because p∵p 是素数 ⇒((p−1)!,p)=1\Rightarrow ((p-1)!, p) = 1⇒((p−1)!,p)=1 (阶乘与 ppp 互质)
    • 两边消去公因子 (p−1)!(p-1)!(p−1)!
    • ∴ap−1≡1(modp)\therefore a^{p-1} \equiv 1 \pmod p∴ap−1≡1(modp)

欧拉函数

为了把费马小定理推广到任意整数（不仅仅是素数），我们需要先定义一个计数工具：欧拉函数 ϕ(n)\phi(n)ϕ(n)。

定义

ϕ(n)\phi(n)ϕ(n) 表示在 111 到 nnn 之间，有多少个正整数与 nnn 互质。

计算公式

若 nnn 的标准分解式为 n=p1a1p2a2…pkakn = p_1^{a_1} p_2^{a_2} \dots p_k^{a_k}n=p1a1​​p2a2​​…pkak​​，则：

ϕ(n)=n(1−1p1)(1−1p2)…(1−1pk)\phi(n) = n \left(1 - \frac{1}{p_1}\right) \left(1 - \frac{1}{p_2}\right) \dots \left(1 - \frac{1}{p_k}\right)ϕ(n)=n(1−p1​1​)(1−p2​1​)…(1−pk​1​)

例子

  • ϕ(10)\phi(10)ϕ(10)：与 10 互质的数有 1,3,7,91, 3, 7, 91,3,7,9，共 4 个。所以 ϕ(10)=4\phi(10)=4ϕ(10)=4。
  • ϕ(p)\phi(p)ϕ(p)：如果 ppp 是素数，除了它自己，比它小的都跟它互质，所以 ϕ(p)=p−1\phi(p) = p-1ϕ(p)=p−1。

欧拉定理

这是费马小定理的威力加强版（通用版）。

定理描述

设 nnn 为大于 1 的正整数，若整数 aaa 与 nnn 互质（即 (a,n)=1(a, n) = 1(a,n)=1），则：

aϕ(n)≡1(modn)a^{\phi(n)} \equiv 1 \pmod naϕ(n)≡1(modn)

💡 核心直觉

不管模数 nnn 是什么，只要底数 aaa 和 nnn 互质，那么 aaa 的指数每经过 ϕ(n)\phi(n)ϕ(n) 这么长，余数就会回到 111。

它的证明逻辑实际上是费马小定理证明的“完全推广版”。如果你看懂了刚才费马小定理的证明，这个证明的核心思路是一模一样的，只是把“111 到 p−1p-1p−1”换成了“与 nnn 互质的数”。

定理描述： 设 nnn 为正整数，若整数 aaa 与 nnn 互质（即 (a,n)=1(a, n) = 1(a,n)=1），则：

两个定理的关系

费马小定理只是欧拉定理的一个特例。

  • 欧拉定理说：周期是 ϕ(n)\phi(n)ϕ(n)。
  • 当 nnn 是素数 ppp 时：ϕ(p)=p−1\phi(p) = p-1ϕ(p)=p−1。
  • 代入公式就变成了费马小定理：ap−1≡1(modp)a^{p-1} \equiv 1 \pmod pap−1≡1(modp)。

🚀 实际应用：降幂大法

这两个定理最强大的用处是简化超级大的指数。 计算 ab(modn)a^b \pmod nab(modn) 时，我们可以把指数 bbb 对 ϕ(n)\phi(n)ϕ(n) 取模，从而把巨大的 bbb 变小。

公式：

ab≡ab(modϕ(n))(modn)a^b \equiv a^{b \pmod{\phi(n)}} \pmod nab≡ab(modϕ(n))(modn)

总结

  • 费马小定理：素数的特殊性质，周期是 p−1p-1p−1。
  • 欧拉定理：任意整数的通用性质，周期是 ϕ(n)\phi(n)ϕ(n)。
  • 欧拉函数：计算互质数的个数，是两个定理的关键工具。

这两个定理不仅是数论的精华，也是现代密码学的数学基础。它们揭示了**“指数循环”**的规律，让我们能够在模运算中安全地处理大数。

一次同余方程有解的充要条件

  1. 一次同余方程 ax≡b(modn)ax \equiv b \pmod nax≡b(modn) 有解的充要条件是 d∣bd \mid bd∣b，其中 d=(a,n)d = (a, n)d=(a,n)。
  2. 一次同余方程 ax≡b(modn)ax \equiv b \pmod nax≡b(modn) 恰有 (a,n)(a,n)(a,n) 个解

证明过程

  1. 转化 (同余 →\to→ 等式)

    • 根据同余定义，方程 ax≡b(modn)ax \equiv b \pmod nax≡b(modn) 等价于存在整数 yyy，使得：
