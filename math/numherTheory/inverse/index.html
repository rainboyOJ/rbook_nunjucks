概念

模逆元是一个数在模意义下的逆元，即对于一个整数 aaa 和模数 mmm，如果存在一个整数 xxx，使得 a×x≡1(modm)a \times x \equiv 1 \pmod{m}a×x≡1(modm)，那么 xxx 就是 aaa 在模 mmm 下的逆元，记作 a−1a^{-1}a−1。

逆元存在的 充要条件 是 aaa 和 mmm 互质，即 gcd⁡(a,m)=1\gcd(a, m) = 1gcd(a,m)=1。

证明过程见 一次同余方程有解的充要条件

概念2

证明: 除以一个数取模等于乘以这个数的逆元取模

ab≡a×b−1(modm) \frac{a}{b} \equiv a \times b^{-1} \pmod{m} ba​≡a×b−1(modm)

这是为您准备的证明，采用极简算式推导流。

前提条件：bbb 与模数 mmm 互质（即 gcd⁡(b,m)=1\gcd(b, m) = 1gcd(b,m)=1），这样 bbb 的逆元 b−1b^{-1}b−1 才存在。

证明：模除法转化为乘逆元

  1. 定义 (将除法转化为乘法方程)：

    • 设 xxx 为 aaa 除以 bbb 的结果：
    • x≡ab(modm)x \equiv \frac{a}{b} \pmod mx≡ba​(modm)
    • 使用同余相乘的运算性质,两边同时乘以 bbb：
    • b⋅x≡a(modm)b \cdot x \equiv a \pmod mb⋅x≡a(modm)

  2. 操作 (两边同乘逆元)：

    • 在等式两边同时乘以 bbb 的逆元 b−1b^{-1}b−1：
    • b−1⋅(b⋅x)≡b−1⋅a(modm)b^{-1} \cdot (b \cdot x) \equiv b^{-1} \cdot a \pmod mb−1⋅(b⋅x)≡b−1⋅a(modm)

  3. 结合 (利用乘法结合律)：

    • (b−1⋅b)⋅x≡a⋅b−1(modm)(b^{-1} \cdot b) \cdot x \equiv a \cdot b^{-1} \pmod m(b−1⋅b)⋅x≡a⋅b−1(modm)

  4. 消去 (利用逆元定义)：

    • 因为 b−1⋅b≡1(modm)b^{-1} \cdot b \equiv 1 \pmod mb−1⋅b≡1(modm)：
      • x≡x(modm)x \equiv x \pmod mx≡x(modm)
      • (b−1⋅b)⋅x≡1⋅x(modm)≡x(modm) \begin{align*} (b^{-1} \cdot b) \cdot x &\equiv 1 \cdot x \pmod m \\ &\equiv x \pmod m \end{align*} (b−1⋅b)⋅x​≡1⋅x(modm)≡x(modm)​
    • 1⋅x≡a⋅b−1(modm)1 \cdot x \equiv a \cdot b^{-1} \pmod m1⋅x≡a⋅b−1(modm)

  5. 结论：

    • x≡a×b−1(modm)x \equiv a \times b^{-1} \pmod mx≡a×b−1(modm)

证毕

  1. 证明反复使用了 同余的运算性质: 同余相乘.
  2. 直觉: 除以一个数等于乘以这个数的逆元(倒数).

        // !! 🧠记忆口诀 “递归反着传 (y, x)，回来减乘除 (y -= a/b*x)”
// 求解 ax + by = gcd(a, b)
// 返回值为 gcd(a, b)
template<typename T = long long>
T exgcd(T a, T b, T &x, T &y) {
    if (!b) {
        x = 1, y = 0;
        return a;
    }
    // 【核心技巧】注意这里的传参顺序：y, x
    // 这样递归回来后：x 存的是上一层的 y'，y 存的是上一层的 x'
    T d = exgcd(b, a % b, y, x);
    
    // 公式直接简化为一行：y = x' - (a/b) * y'
    // 此时当前的 y 已经是 x'，当前的 x 已经是 y'
    y -= a / b * x; 
    
    return d;
}

// 求 a 在模 m 下的逆元 (前提: gcd(a, m) = 1)
template<typename T = long long>
T inv(T a, T m) {
    T x, y;
    exgcd(a, m, x, y);
    // 使用 (x % m + m) % m 确保结果为正数
    return (x % m + m) % m;
}

线性求逆元

学习“线性求逆元”算法，重点在于理解它的应用场景和那个神奇的递推公式。

为了让你心智负担最低，我把它拆解为三个步骤：为什么用、公式怎么来的、代码怎么写。

如果题目要你求 1,2,3,…,n1, 2, 3, \dots, n1,2,3,…,n 所有数的逆元呢？

  • 用 exgcd 算 nnn 次：总耗时 O(nlog⁡p)O(n \log p)O(nlogp)。如果 nnn 是 10710^7107 级别，会超时。
  • 用 线性算法：总耗时 O(n)O(n)O(n)。平均求一个逆元只要 O(1)O(1)O(1) 的时间！

结论：当你需要批量计算一堆逆元（比如求组合数 C(n,m)%pC(n, m) \% pC(n,m)%p 需要用到很多阶乘的逆元）时，必须用线性算法。

极简推导流 (公式是怎么来的)

设模数为 ppp，我们需要求 iii 的逆元 i−1i^{-1}i−1。

  1. 带余除法：

    • 令 p=k⋅i+rp = k \cdot i + rp=k⋅i+r
    • 其中 k=⌊pi⌋k = \lfloor \frac{p}{i} \rfloork=⌊ip​⌋ (商)，r=p mod ir = p \bmod ir=pmodi (余数)。

  2. 列同余式：

    • k⋅i+r≡0(modp)k \cdot i + r \equiv 0 \pmod pk⋅i+r≡0(modp)
    • (因为 k⋅i+rk \cdot i + rk⋅i+r 等于 ppp，而 ppp 模 ppp 是 0)

  3. 神奇操作 (两边同时乘 i−1⋅r−1i^{-1} \cdot r^{-1}i−1⋅r−1 )：

    • k⋅r−1+i−1≡0(modp)k \cdot r^{-1} + i^{-1} \equiv 0 \pmod pk⋅r−1+i−1≡0(modp)

  4. 移项求 i−1i^{-1}i−1：

    • i−1≡−k⋅r−1(modp)i^{-1} \equiv -k \cdot r^{-1} \pmod pi−1≡−k⋅r−1(modp)
    • i−1≡−⌊pi⌋×(p mod i)−1(modp)i^{-1} \equiv -\lfloor \frac{p}{i} \rfloor \times (p \bmod i)^{-1} \pmod pi−1≡−⌊ip​⌋×(pmodi)−1(modp)

  5. 代回变量：

    • kkk 替换为 ⌊pi⌋\lfloor \frac{p}{i} \rfloor⌊ip​⌋
    • rrr 替换为 p mod ip \bmod ipmodi
    • −k-k−k 在模运算中写成 p−kp - kp−k 以保证正数

  6. 边界

  • 1−1≡1(modp)1^{-1} \equiv 1 \pmod p1−1≡1(modp)
  • 当 i=1i=1i=1 时，逆元就是 111。

最终公式：

inv[i]=(p−⌊pi⌋)×inv[p mod i]%pinv[i] = (p - \lfloor \frac{p}{i} \rfloor) \times inv[p \bmod i] \% pinv[i]=(p−⌊ip​⌋)×inv[pmodi]%p

这个公式说明: 一个数字的逆元等于它的商的负数乘以它的余数的逆元。

i−1≡−⌊pi⌋×(p mod i)−1(modp) i^{-1} \equiv -\lfloor \frac{p}{i} \rfloor \times (p \bmod i)^{-1} \pmod pi−1≡−⌊ip​⌋×(pmodi)−1(modp)
  • 一个数字的逆元：i−1i^{-1}i−1
  • 等于：≡\equiv≡
  • 它的商的负数：−⌊pi⌋-\lfloor \frac{p}{i} \rfloor−⌊ip​⌋
  • 乘以：×\times×
  • 它的余数的逆元：(p mod i)−1(p \bmod i)^{-1}(pmodi)−1

时间复杂度: O(log2P)O(log2P)O(log2P) ,证明略.

        // 线性求 1 到 n 所有数的逆元
// 时间复杂度 O(n)
#include <iostream>
#include <vector>
using namespace std;

long long inv[2000005]; // 数组开到最大范围
long long p; // 模数

void init_inverse(int n) {
    inv[1] = 1; // 【奠基】1的逆元是1
    
    for (int i = 2; i <= n; ++i) {
        // 【核心公式】
        // inv[i] = (p - p/i) * inv[p % i] % p
        inv[i] = (p - p / i) * inv[p % i] % p;
    }
}

int main() {
    int n;
    cin >> n >> p;
    init_inverse(n);
    
    // 输出验证
    for(int i=1; i<=n; i++) {
        cout << inv[i] << "\n";
    }
    return 0;
}

🧠 记忆钩子

怎么记住 (p - p / i) * inv[p % i] 这一长串？

  1. 结构：它是 “商” ×\times× “余数的逆元”。
  2. 符号：因为移项时有个负号，所以用 p - ... 来处理负数。
  3. 依赖：我要算 iii 的逆元，我得问问比我小的 p%ip \% ip%i 的逆元是多少。
